email -> sigla para eletronic email - correio eletrônico

host -> Espécie de servidor
HTML -> linguagem de marcação da internet

login/logon -> é o termo que é usado para entrar em um site
logout/logoff -> desligar

firewall -> parede cuja função é impedir acesso indevido no seu computador

fake news -> notícias falsas que se passam por verdade, boatos
